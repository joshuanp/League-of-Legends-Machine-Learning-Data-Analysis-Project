# League-of-Legends-Machine-Learning-Data-Analysis-Project
Performing ML modeling techniques to predict a winner from the first 10 minutes of League of Legends games.

## Methods Used

* Data Cleaning
* Exploratory Data Analysis
* Machine Learning
* Logistic Regression
* Elastic Net
* Classification Tree
* Random Forest

## Technologies

* R
* Tidymodels
* ggplot2
* R Markdown
